IP drivers: Common drivers used on multiple NXP devices

Detailed Description

NXP devices share common peripherals across multiple devices. IP drivers are common drivers that can be used with one or more devices. IP drivers are configured at build-time for a specific device by the use of CHIP_* definitions or optional feature (ie, USE_RMII) definitions.
