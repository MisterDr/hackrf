CHIP: LPC17xx/40xx EMC Driver
LPC17xx/40xx chip specific drivers

Detailed Description

The EMC interface clocks must be enabled outside this driver prior to calling any function of this driver with the Chip_Clock_EnablePeripheralClock() function.
