Mass Storage Host
USB examples

Detailed Description

Example description
This example implements a mass storage class host mode device that enumerates a mass storage class device (USB flash drive). It reads the first sector of the device and displays a hexdump style listing of that data on a terminal.

When the example is first run the terminal window will display: Mass Storage Host Demo running.

When a flash drive is plugged in the terminal window will display something similar to this:
