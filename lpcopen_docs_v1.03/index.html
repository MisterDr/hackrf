The LPCOpen technical library contains a number of different topics related to LPCOpen project development, board and project porting, design approach, etc.
LPCOpen technical library

a LPC Driver Library for NXP microcontrollers
  • IP Driver Layer IP specific low-level drivers- can be used alone
  • Chip Driver Layer Chip specific drivers, may use IP drivers
  • Board Support Layer Board support code for various support boards
Lots of supported software packages, libraries, and stacks
  • Supported RTOSes such as FreeRTOS and uC/OS-III
  • Supported Graphics packages such as emWin and SWIM
  • Supported networking stacks such as LWIP
  • Supported USB stacks such as LPCUSBLIB
  • Supported file systems such as Chan the FAT filesystem
