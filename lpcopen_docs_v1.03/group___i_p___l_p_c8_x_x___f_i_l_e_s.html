CHIP: LPC8XX Chip layer required IP layer drivers
LPC8xx chip specific drivers

This is a list of the IP drivers required for the LPC8XX device family.
(acmp_001.c, acmp_001.h) IP: Analog comparator driver
(crc_001.c, crc_001.h) IP: CRC register block and driver
(gpio_001.h) IP: GPIO register block and driver
(mrt_001.h) IP: MRT register block and driver
(sct_001.c, sct_001.h) IP: SCT register block and driver
(spi_002.c, spi_002.h) IP: SPI register block and driver (002)
(usart_002.c, usart_002.h) IP: USART2 register block and driver
(wkt_001.h) IP: Self Wakeup Timer (WKT) register block and driver
(wwdt_001.c, wwdt_001.h) IP: WWDT register block and driver
